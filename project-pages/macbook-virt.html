How to run VMs on your macbook
As part of my work, I deal with a lot of complex scenarios involving virtual machines. I also might need to write and experiment with code on actual linux systems, and having a local setup linux VM comes in handy. In this blog, I will talk you through how to setup a ubuntu VM on MacOS with UTM.
Pre requisites
First, download and install UTM for macos : https://mac.getutm.app/
Once installed, you need an ubuntu ISO image compatible with M1 macs. I like to use ubuntu 20.04 arm version which you can find here.
Download ubuntu-20.04.5-live-server-arm64.iso
